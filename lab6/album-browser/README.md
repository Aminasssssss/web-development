# Album Browser

Single-page application for browsing albums and photos using Angular and JSONPlaceholder API.

## Features

- View all albums in responsive grid
- View album details with editable title
- Delete albums
- View album photos in grid layout
- Edit album titles
- Loading states and error handling
- Navigation with active link highlighting
- Lazy loading for better performance
- Real photos from Picsum
- Russian titles for better UX

## Technologies

- Angular 21
- Angular Router
- HttpClient
- RxJS Observables
- Signals for state management
- JSONPlaceholder API
- Picsum Photos API

## Prerequisites

- Node.js (v18 or higher)
- Angular CLI (v21 or higher)
